# 本地线程学习

## 本地线程

本地线程： 是解决线程或协程独立保存数据存在的

以前我们学习了多进程，多线程

进程间数据是隔离的， 要想实现数据交互，方法有： rpc,Queue，socket,Array,Manager
	from multiprocessing import Process,Array,Manager

线程间数据是共享的
```
import time
import threading

class Foo():
    def __init__(self):
        pass

local_values = Foo()

def task(num):
    local_values.name = num  # 由于线程数据共享，线程后改的数据会覆盖前改的数据
    time.sleep(1)
    print(local_values.name,threading.current_thread().name)

for i in range(1,6):
    t = threading.Thread(target=task, args=(i,), name='线程名%s'%i)
    t.start()
```

打印
```
5 线程名1
5 线程名3
5 线程名2
5 线程名5
5 线程名4
```

线程间数据是共享的要想实现隔离，方法有： threading.Threadlocal()本地线程，来看下面这段代码实现独立保存数据
```
import time
import threading

local_values = threading.local()

def task(num):
    local_values.name = num  # 由于线程数据共享，线程后改的数据会覆盖前改的数据
    time.sleep(1)
    print(local_values.name,threading.current_thread().name)

for i in range(1,6):
    t = threading.Thread(target=task, args=(i,), name='线程名%s'%i)
    t.start()
```

打印
```
1 线程名1
2 线程名2
4 线程名4
3 线程名3
5 线程名5
```
